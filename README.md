# Portfólio de Exercícios em Python

Este repositório contém exercícios desenvolvidos em Python com foco em lógica de programação e prática de conceitos básicos.

## 📌 Exercícios:

### 🧮 Exercício 01
Cálculo da área e perímetro de um retângulo.

### 💰 Exercício 02
Cálculo de salário com acréscimo de comissão de 5%.

### 🚗 Exercício 03
Cálculo de velocidade média com base na distância e tempo.

### 💵 Exercício 05
Conversão de dólar para real.

### 🍽️ Exercício 06
Cálculo de valor total com taxa de serviço de 10%.

## 🧠 O que foi praticado:
- Entrada de dados (input)
- Operações matemáticas
- Lógica de programação
- Saída de dados (print)

## 🎯 Objetivo:
Desenvolver habilidades em programação e melhorar o raciocínio lógico por meio de exercícios práticos.
